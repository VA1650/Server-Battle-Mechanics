# Simple-Games-Collection

## ⚔️ Server Battle Mechanics (Prototype)

Консольная пошаговая стратегия, моделирующая атаку и защиту серверных узлов. Проект демонстрирует реализацию игровой логики без использования внешних движков.
Technical Highlights:

    Resource Management: Система распределения «мощности» (power) между тремя узлами.

    State Tracking: Отслеживание состояния объектов (HP/Power) в реальном времени.

    Damage Logic: Реализация механики таргетированной атаки (выбор конкретного узла оппонента).

Игровая логика:

У каждого игрока есть стек из серверов (в данном случае 3 узла по 100 единиц мощности)

    Игрок выбирает индекс целевого сервера.

    Система рассчитывает урон и обновляет состояние пула серверов.

    Побеждает тот, кто первым обнулит суммарную мощность противника.

    Note on AI: Текущая версия бота использует базовый рандомизатор. Это отличный полигон для реализации более сложных алгоритмов, например, Minimax или эвристического анализа самого «просевшего» узла игрока.
