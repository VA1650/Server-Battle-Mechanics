# Server Warfare: Console Strategy ⚔️

Пошаговая тактическая игра, реализованная на C++. Проект начался как простая игра в 2021 году и эволюционировал в демонстрацию базовых принципов игрового дизайна и программирования ИИ.
🚀 Эволюция проекта

## В репозитории сохранена история коммитов, отражающая переход от линейного процедурного кода к структурированной архитектуре:

    v1.0 (Legacy): Базовая логика внутри main(), использование rand().

    v2.0 (Refactored): Внедрение структур данных, декомпозиция функций и защита от некорректного ввода.

    v2.1 (Smart AI): Реализация эвристического ИИ с памятью ходов и визуализация через ANSI-цвета.

## 🧠 Особенности ИИ

от в игре — это не просто генератор случайных чисел. Он использует:

    Lethal Range Priority: Поиск узлов, которые можно уничтожить одним ходом.

    Counter-Defense Logic: Бот запоминает последний защитный ход игрока и с вероятностью 80% атакует незащищенные узлы.

    Smart Fortification: Приоритизация защиты собственных наиболее уязвимых серверов.

## 🛠 Технологии

    Стандарт: C++17 (использование Structured Bindings для возврата ходов ИИ).

    Интерфейс: CLI с использованием ANSI-последовательностей для цветовой индикации событий.
